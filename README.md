# Advanced Hand Gesture 3D Visualizer

## What is this project?
This project is an **Advanced Hand Gesture 3D Visualizer** that uses your webcam and hand tracking to interactively render and manipulate a glowing 3D tree in real time. It leverages computer vision and machine learning libraries to detect hand gestures, allowing you to reveal, rotate, and grow a virtual tree using intuitive hand movements. The visualization is highly optimized for performance and features beautiful bloom effects and particle animations.

- **Technologies used:** Python, OpenCV, MediaPipe, NumPy
- **Features:**
  - Real-time hand gesture recognition
  - 3D tree generation and rendering
  - Interactive controls for growing, rotating, and exploding the tree
  - Visual effects: bloom, glow, and particles

## How to fork this project
1. **Fork on GitHub:**
   - Go to the repository page on [GitHub](https://docs.github.com/en/get-started/quickstart/fork-a-repo) and click the **Fork** button in the top-right corner.
   - This will create a copy of the project under your own GitHub account.
2. **Clone your fork:**
   - Copy the URL of your forked repository.
   - Open a terminal and run:
     ```bash
     git clone https://github.com/your-username/your-forked-repo.git
     ```
3. **Install dependencies:**
   - Make sure you have Python 3 installed.
   - The script will auto-install required packages (`opencv-python`, `mediapipe`, `numpy`) on first run.
4. **Run the project:**
   - Navigate to the project directory and run:
     ```bash
     python Run.py
     ```
